# Noda Frontend

Vue.js + Vite + Pinia + Vue Router

## เริ่มต้น

```bash
npm install
# ตั้งค่า .env.development
npm run dev
```

## .env.development
```
VITE_API_URL=http://localhost:3000
VITE_COMMUNITY_ID=community-a
VITE_SKIP_AUTH=false
```

## Deploy บน Firebase
```bash
npm run build
firebase deploy
```
